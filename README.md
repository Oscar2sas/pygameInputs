# 🎮 Pygame UI Interface Kit

Una librería modular y ligera para crear interfaces gráficas de usuario (GUI) en **Pygame** utilizando Programación Orientada a Objetos (POO). 

Este kit abstrae la complejidad de manejar eventos y dibujo de formas, permitiéndote crear botones, cuadros de texto y selectores con solo un par de líneas de código.

---

## ✨ Características principales

- 🔘 **Botones Dinámicos:** Auto-ajuste de tamaño, efectos de *hover* (pasar el mouse) y sombras.
- ✍️ **Input Text:** Manejo de enfoque y entrada de teclado integrada.
- 🔘 **Radio Groups:** Sistema de selección única para opciones múltiples.
- 🎚️ **Componentes Extra:** Sliders y Checkboxes listos para usar.
- 🛠️ **Modular:** Fácil de integrar en cualquier bucle `while` de Pygame.
